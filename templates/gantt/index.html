    Planificare Gantt automata din F3

    Incarca o lista de cantitati / deviz (F3) si genereaza automat structura de planificare: WBS ierarhic, activitati, durate, lant de dependente tehnologice (FS/SS/FF/SF + decalaj) si export catre MS Project / Primavera P6 / CSV. Fara legare manuala a predecesorilor.

    1. Import XLSX / CSV, 100k+ randuri
    2. Clasificare tehnologica (sapatura, pozare...)
    3. WBS Obiect → Tronson → Categorie
    4. Dependente replicate pe fiecare tronson
    5. Validare cicluri, orfani, erori
    6. Export P6 / MS Project / CSV
    Regulile (clasificare, dependente, randamente) sunt configurabile in config/gantt/*.json.
